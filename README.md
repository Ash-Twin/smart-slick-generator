# smart-slick-generator
> semi-auto slick table generator

## How to use
1. download release binary
2. `tar -zxvf generator.tar.gz`
3. `cd stage/bin`
4. `./smart-slick -f your_ddl.sql`
5. new scala file will be created. Have fun.
